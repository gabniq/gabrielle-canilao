Play A Totally Normal Pong Game!

For this project, I really wanted to lean into developing different game UI. The whole game was coded in p5js using Javascript rather than any game engine, so all UI and mechanics were coded by hand.

The game starts as the classic game of Pong, then leans into different styles of game, transitioning into a visual novel style layout and then a Pokemon-esque fight scene. In creating A Totally Normal Pong Game, I referenced three p5jslibraries: the sound library, scenemanager, and p5play. I really wanted to challenge myself for this project as I allowed myself a large timeframe to work on this, so I took my time doing research and planning out my coding in order to complete these three completely different GUIs and game genres that are cycled through.

I also wanted to exemplify theatrification, as my project takes something seemingly simple--pong--and turns it on its head as it heads into different gamestyles of vastly different genres. Additionally, though it does follow a pong ball, my project mashes up genres that are entirely different, but still highly recognizable by its UI and artstyle.
