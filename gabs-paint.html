Run Gab's Paint!

For this project, I was very excited to create brushes, as it was something I had done a couple of times before using the built-in feature on my normal drawing program. I wanted to recreate some brushes that I typically use myself, including those that could help recreate both a manga-like and painterly style. The whole program was created on p5js using Javascript. When creating the program, I had wanted to explore many different features of Javascript, from using clipping masks to sliders to for loops in order to ensure unique brushes.

Overall, I was highly successful in creating what I set out to achieve. The halftone brush in particular is something that I am proud of, as it accurately recreates the one that I often use. Additionally, the effort I had spent into creating a color-picking system is a success, as color is important to me in my own work and I did not want to limit my palette. It was actually quite challenging to do so, as I was not as familiar with Javascript as I am now when originally creating the program. As such, I ended up doing a bit of research into learning how to create it; though, rather than the frustration that I often get when running into problems, I actually found myself quite capitvated by trying to figure it out, taking the challenge in stride like it was a puzzle. Considering how often my own art style changes, I wanted to include many brushes that reflect that. In particular, I wanted to be able to produce both painter-esque and manga-style works, as seen in the example works above, which were all created in the program.
