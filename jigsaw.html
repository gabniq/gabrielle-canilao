View Jigsaw Here!

As I often do in my practice, for this animation, I chose to combine music with art. In creating Jigsaw, I knew that I wanted to video to have the background music of Radiohead's "Jigsaw Falling Into Place," one of my favorite songs. To build off of this, the visuals I used were inspired by the cover art of the album.

In order to create Jigsaw, the programmed animation was developed using p5js code, taking pre-existing algorithm and remixing it put my own spin on it and create something unique. The two algorithms I used were Brownian Motion and Kaleidoscope. In order to fit my needs, I changed the shapes/images of the original Brownian Motion code within my class containers, as well as added variables to change the color. Overall, I wanted to create smooth transitions between each "phase" while keeping it in theme with the song (hence the jigsaw pieces in the middle phase), making everything fit together well. One particular difficulty I ran into was my issues in exporting each frame, as my computer slowed down when downloading frames for the second phase, but this was solved through lowering the frame rates for each section and giving myself ample time to run the program and download each frame.
